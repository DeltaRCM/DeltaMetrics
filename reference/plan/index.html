Planview operations¶

The package makes available functions relating to planview operations on data.

The functions are defined in deltametrics.plan.

a_land_function(mask)

Compute a land-water function

compute_delta_radius(mask)

Compute the delta radius

a_channel_function(mask)

Compute a channel function

compute_shoreline_angles(mask[, param2])

Compute shoreline angle.
