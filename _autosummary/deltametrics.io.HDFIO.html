deltametrics.io.HDFIO¶

class deltametrics.io.HDFIO(data_path)¶

Utility for consistent IO with HDF5 files.

This module wraps calls to the hdf5 python module in a consistent API, so the user can work seamlessly with either HDF5 files or netCDF4 files. The public methods of this class are consistent with NetCDFIO.
