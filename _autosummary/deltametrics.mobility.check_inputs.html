deltametrics.mobility.check_inputs¶

deltametrics.mobility.check_inputs(chmap, basevalues, time_window, landmap=None)¶

Check the input variable values.

Ensures compatibility with mobility functions. Tries to convert from some potential input types (xarray data arrays, deltametrics masks) if possible.

Parameters

  • chmap (ndarray) – A t-x-y shaped array with binary channel maps.

  • basevalues (list) – List of t indices to use as the base channel maps.

  • time_window (int) – Number of time slices (t indices) to use as the time lag

  • landmap (ndarray, optional) – A t-x-y shaped array with binary land maps. Or a x-y 2D array of a binary base mask that contains 1s in the region to be analyzed and 0s elsewhere.
