deltametrics.cube.Cube¶

class deltametrics.cube.Cube(*args)¶

Data cube object.

Data cube object that contains the x-y-z(-t) information. It may have any number of attached attributes (grain size, mud frac, elevation). Time is assumed to be constant, unless specified otherwise.

Note

This object is going to be core to the success of the project.
