deltametrics.cube.CubeVariable¶

class deltametrics.cube.CubeVariable(xarray_obj)¶

Slice of a Cube.

Slicing an Cube returns an object of this type. The CubeVariable is an accessor of the xarray.DataArray class, enabling additional functions to be added to the object, while retaining the xarray object and it’s native functionality under CubeVariable.data.

Warning

You probably should not instantiate objects of this type directly.

Examples

>>> import deltametrics as dm
>>> rcm8cube = dm.sample_data.rcm8()

>>> type(rcm8cube['velocity'])
<class 'deltametrics.cube.CubeVariable'>

>>> type(rcm8cube['velocity'].data)
<class 'xarray.DataArray'>

>>> rcm8cube['velocity'].variable
'velocity'

__getitem__(slc)¶

Get items from the underlying data.

Takes a numpy slicing style and slices data from the underlying data. Note that the underlying data is stored in an xarray.DataArray, and this method returns a xarray.DataArray.

Parameters

slc (a numpy slice) – A valid numpy style slice. For example, [10, ...]. Dimension validation is not performed before slicing.
